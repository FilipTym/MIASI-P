# Backend Baseline

This folder contains the Java + ANTLR backend baseline.

## Current layout

- `src/main/antlr` — grammar files (`*.g4`)
- `src/main/java` — handwritten Java sources (`Main.java`, `Visitor.java`)
- `target/generated-sources/antlr4` — generated ANTLR artifacts (build output)

## Notes

- Parser/lexer classes are generated by Maven (`antlr4-maven-plugin`).
- Do not edit generated files under `target/`.

## ANTLR rebuild roadmap

From `backend/`:

1. Regenerate ANTLR sources only:
	- `mvn generate-sources`
2. Full clean + regenerate + compile:
	- `mvn clean compile`
3. (Optional) package backend:
	- `mvn package`

If you change `ExprLexer.g4` or `ExprParser.g4`, run `mvn clean compile` to refresh generated classes and rebuild.
